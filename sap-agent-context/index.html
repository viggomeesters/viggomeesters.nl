Project notes

What it is

A public knowledge/context repo that packages SAP concepts and delivery context in a way AI agents can reuse without guessing.

Why it exists

SAP work has lots of implicit field, process, and integration knowledge. The project turns that into explicit, source-labelled context.

Useful for

Functional designs, field mapping, SAP object explanation, migration prep, and agent-assisted delivery workflows.

Design bias

Evidence first: keep generated answers grounded in sources, examples, and reusable context rather than model memory.

This page is the human-readable project entrypoint. GitHub remains the source for code, releases, issues, and the latest technical details.
