How to read this timeline

Each stage is a public explanation of a workflow layer, not a full changelog. The pages name the shift, the useful output, and the limitation that pushed the next stage.

Taxonomy boundary

Agent Workflow is a system evolution page. The reusable rule for running agent work lives under Methodologies as Agent Workflow Loop; individual procedures live under Skills.

Evidence standard

Public copy stays high-level: no private vault contents, no fake metrics, and no claim that a stage was perfect just because it existed.
